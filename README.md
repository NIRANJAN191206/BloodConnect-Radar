# 🩸 BloodConnect Radar

**BloodConnect Radar** is a real-time emergency blood donation matching system. It bridges the gap between donors and receivers using location-based mapping and automated SMS alerts.



## 🚀 Key Features

- **Animated Landing Page**: A smooth, professional entrance with custom CSS animations.
- **Live Donor Radar**: Visualizes donor locations on an interactive map using Leaflet.js.
- **Instant SMS Notifications**: Automatically triggers Twilio SMS alerts to matching donors when an emergency request is raised.
- **Smart Filtering**: Filters donors by blood group and proximity (Chennai, Puducherry, etc.).

## 🛠️ Tech Stack

- **Backend**: Python (Flask)
- **Database**: SQLite3
- **Frontend**: HTML5, CSS3 (Modern UI), JavaScript
- **Maps**: Leaflet.js & OpenStreetMap
- **Communication**: Twilio SMS API
